# Bibliography

This page lists all bibliographic sources cited in the language JSON data files,
grouped by language family. Sources are referenced via the `sources` array in each
language's JSON spec (see `data/SCHEMA.md` for the schema).

---

## Germanic

- **wells1982_vol2** — Wells, J.C. (1982). *Accents of English, Vol. 2: The British Isles*. Cambridge University Press.
- **wells1982_vol3** — Wells, J.C. (1982). *Accents of English, Vol. 3: Beyond the British Isles*. Cambridge University Press.
- **roach2009** — Roach, P. (2009). *English Phonetics and Phonology: A Practical Course* (4th ed.). Cambridge University Press.
- **gimson2014** — Cruttenden, A. (ed.) (2014). *Gimson's Pronunciation of English* (8th ed.). Routledge.
